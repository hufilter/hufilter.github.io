# Operációs rendszer szintű blokkolók

A szűrés az operációs rendszer szintjén történik, és hatással van minden programra, ami az adott operációs rendszeren
fut. Ezek a megoldások helyettesítik a böngészőkbe telepíthető verziókat, így nem kell minden böngészőhöz külön-külön
telepíteni reklámblokkolót. A rendszerszintű blokkolók általában fizetősek, de a legtöbbnek van ingyenes próbaverziója
is, amely időszak alatt el lehet dönteni, hogy megéri-e számodra a fizetős verzió.

    <li><AdBlockUltimateLogo /> <a href="https://adblockultimate.net/windows">AdBlocker Ultimate</a></li>
  </ul>
  
  <li><AndroidLogo /> Android:</li>

</ul>

:::tip
A rendszerszintű blokkolók extra funkciókat is kínálnak, amelyekkel tovább növelhető a biztonság, pl. a DNS
kiszolgáló cseréje, tűzfal funkció, stb.
:::

:::note
Elképzelhető, hogy bizonyos alkalmazásokat nem tudnak érdemben szűrni a rendszerszintű blokkolók, pl. ilyen Androidon
a YouTube alkalmazás. Ennek okáról és a megoldásról bővebben olvashatsz a
[Reklámmentes YouTube Androidon](../mobile-devices.mdx#reklámmentes-youtube-androidon) fejezetben.
:::
